Gerenciador de Contatos
Contact Manager é um sistema desenvolvido em ASP.NET Core MVC para gerenciamento de usuários e contatos, com funcionalidades como autenticação, recuperação de senha e controle de acesso. Utiliza Entity Framework Core e SQL Server, com segurança reforçada por hashing de senhas.
